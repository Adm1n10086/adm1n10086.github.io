# 熟悉常见外网打点技术及内外攻击技术

（一）资产测绘与信息搜集（前置打点基础）
外网打点技术，资产测绘与信息搜集
域名资产探测
子域名爆破、DNS 解析查询、旁站 / C 段查询、备案反查、证书透明度 CT 日志爬取、二级 / 三级域名遍历。
端口 & 服务测绘
Masscan、Nmap 全端口扫描，识别公网开放端口：80/443 (Web)、22 (SSH)、3389 (RDP)、3306 (MySQL)、5432 (Postgres)、6379 (Redis)、8080/8443 中间件、VPN (4433/1723)、防火墙管理口。
指纹识别
Web 中间件（Tomcat、WebLogic、JBoss、Nexus、Apache、IIS）、OA/ERP/ 协同系统（泛微、致远、用友、金蝶）、VPN（深信服、天融信、Forti）、路由防火墙、打印机、监控设备、Git/SVN 代码仓库。
OSINT 开源情报
官网公告、招聘、员工邮箱、GitHub/Gitee 泄露源码、网盘泄露配置、第三方平台泄露账号、供应商信息。
（二）Web 服务漏洞打点（最主流外网入口）
通用 Web 漏洞
SQL 注入、XSS 跨站、文件上传、文件包含、目录遍历、SSRF、命令执行、逻辑越权、未授权访问、弱口令后台。
中间件漏洞
WebLogic 反序列化、Tomcat 弱口令 / 后门、Shiro 反序列化、Fastjson、Struts2、Nexus 未授权、Jenkins 命令执行。
专用业务系统漏洞
OA 漏洞、致远 / 泛微任意文件上传 / 命令执行、Exchange ProxyLogon/ProxyShell、Confluence RCE、通达后台漏洞、企业邮箱漏洞。
（三）端口服务弱口令 & 未授权打点
数据库：Redis 未授权访问、MySQL/MSSQL 弱口令、Mongo 未授权、Elasticsearch 未授权漏洞。
远程管理：SSH 弱口令、RDP 爆破、VNC 无密码、WinRM 未授权。
缓存 / 消息队列：Redis 写 Webshell、RabbitMQ 弱口令、Kafka 未授权。
代码仓库：Git/SVN 泄露源码、Gogs/GitLab 弱口令、未授权访问。
（四）边界设备专项打点
SSL VPN/IPSec VPN：弱口令、漏洞 RCE、验证码绕过、短信验证码爆破、Cookie 伪造。
防火墙、负载均衡：管理后台弱口令、命令执行、后台未授权。
路由器、摄像头、打印机：固件漏洞、默认账号、远程命令执行。
（五）钓鱼类外网打点（APT 高频）
鱼叉钓鱼邮件：伪造合作伙伴、法务、招标，宏 Office、恶意 LNK、PDF 漏洞、钓鱼站点。
仿站钓鱼：仿 VPN、OA、企业邮箱登录页窃取账号密码。
社交钓鱼：企业微信、钉钉、LinkedIn 定向发送恶意文件 / 链接。
（六）供应链第三方打点
攻击厂商、外包系统、开源组件投毒，利用企业对外采购的工具、更新服务器实现外网入口。

信息收集的方式

信息收集方式（渗透测试/网络安全通用）
一、公开渠道被动信息收集（Passive，不触碰目标服务器）

  1. 域名&备案信息

  2. WHOIS查询：注册人、手机号、邮箱、注册时间、DNS服务器

  3. 工信部备案：企业名称、法人、备案地址、关联域名

  4. 企查查/天眼查：企业工商信息、股东、分支机构、关联公司、联系方式

  5. DNS信息收集

  • DNS解析记录：A、AAAA、CNAME、MX、TXT、NS记录
  • 子域名挖掘：证书透明度CT日志、搜索引擎、第三方子域名库
  • 反向DNS、DNS区域传送漏洞
  1. 搜索引擎爬取
  • 百度、谷歌、搜狗语法：site、inurl、filetype、intitle
  • 遗留备份文件：.bak、.sql、.zip、.log、后台地址、源码泄露
  • 公开招聘网站：服务器架构、中间件、开发语言、内网设备信息

  1. 第三方平台数据

  2. 网络空间测绘：Shodan、FOFA、Censys，查IP端口、设备型号、banner

  3. 代码托管平台：GitHub/Gitee/GitLab，搜索公司关键词找泄露源码、密钥、数据库地址

  4. 社交平台：微信公众号、微博、知乎、企业群，获取员工姓名、工号、邮箱、内部系统名称

  5. 历史快照：Wayback Machine网页存档，查看下线旧页面、后台路径

  6. 证书、邮件、爬虫数据

  • SSL证书：获取域名列表、企业名称
  • 邮箱搜集：公开新闻、招聘、官网底部，整理员工邮箱用于钓鱼、爆破

二、主动信息收集（Active，直接和目标网络交互）

  1. 主机存活探测
  • ICMP：ping、arping（内网）
  • TCP/UDP端口扫描：Nmap、Masscan，探测开放端口、服务版本、操作系统
  1. 端口与服务识别
  • 抓取Banner：识别Apache、Nginx、Tomcat、MySQL、Redis、FTP等版本
  • 漏洞匹配：根据服务版本对应CVE漏洞
  • 弱口令探测：FTP、SSH、3389、数据库、后台管理端口

  1. Web站点信息探测

  2. 目录扫描：Dirsearch、Dirmap、御剑，扫描后台、上传、备份、管理页面

  3. 指纹识别：CMS（织梦、ThinkPHP、Shiro、SpringBoot）、前端框架、中间件

  4. 爬虫遍历全站：获取接口、参数、页面功能、文件上传点、登录入口

  5. 工具探测：AWVS、Xray、Nessus自动化爬取站点资产

  6. 内网信息收集（拿到权限后）

  • 网段扫描、ARP扫描、路由、网关、DHCP服务器
  • 域信息：域控、域用户、共享文件夹、组策略、内网数据库
  • 本机信息：操作系统版本、进程、服务、账户、浏览器密码、配置文件

三、社工类信息收集（社会工程学）

  1. 公开社交信息：员工朋友圈、抖音、脉脉，收集姓名、生日、手机号、部门
  2. 钓鱼：伪造邮件/二维码/登录页面，诱导输入账号密码、上传文件
  3. 电话社工：伪装运维、快递、厂商套取内网地址、账号、服务器密码
  4. 线下：公司海报、工牌、办公文档、废弃打印纸获取内部信息

四、其他辅助收集手段

  1. 流量抓包：Wireshark、Burp抓HTTP/HTTPS请求，获取接口、Cookie、参数
  2. 流量镜像/日志：服务器访问日志、防火墙日志，分析访问IP、路径
  3. APP/小程序抓包：反编译、抓接口，获取后端域名、API密钥
  4. 第三方泄露库：社工库、泄露数据库，查询泄露账号密码、手机号
